  • Peserta didik mampu menjelaskan fungsi masing-masing perangkat keras.

Materi Hardware Komputer

1. Input Device

Input device adalah perangkat keras untuk memasukkan data ke komputer.

  • Keyboard
  • Mouse
  • Scanner
  • Webcam

2. Processing Device

Processing device adalah perangkat yang mengolah data.

  • CPU
  • RAM
  • Motherboard

3. Output Device

Output device adalah perangkat untuk menampilkan hasil proses komputer.

  • Monitor
  • Printer
  • Speaker

4. Storage Device

Storage device adalah perangkat untuk menyimpan data.

  • HDD
  • SSD
  • Flashdisk
